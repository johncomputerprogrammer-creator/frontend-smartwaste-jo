# SMARTWASTE Frontend

A Flutter-based frontend for the SMARTWASTE smart waste management system.  

## Features
- User login and registration
- Role-based dashboards (Resident, Collector, Admin)
- Responsive UI for web and mobile
- Integrates with SMARTWASTE Backend API

## Technologies
- Flutter
- Dart
- Firebase (if used for auth) or Backend API
